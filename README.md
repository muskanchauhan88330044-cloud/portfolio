# 🚀 Getting Started

## 1. Clone the Repository

```bash
git clone https://github.com/your-github-username/muskan-portfolio.git
```

---

## 2. Open the Project Folder

```bash
cd muskan-portfolio
```

---

## 3. Install Dependencies

```bash
npm install
```

---

## 4. Run the Development Server

```bash
npm run dev
```

---

## 5. Build the Project

```bash
npm run build
```
